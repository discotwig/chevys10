# chevys10 — 1999 Chevy S10 (2.2L I4) owner reference

Personal planning and documentation for a bone-stock **1999 Chevrolet S10** with the **2.2L inline-four**—curated so it can grow into a useful reference for other owners of the same era and engine.

## Mission

This repository is the working home for maintenance plans, how-tos, and gotchas. Content is built by gathering public S10 material—service manuals and bulletins where available, reputable guides, and community knowledge—and **synthesizing** it into short, practical write-ups: step-by-step procedures, common mistakes, torque and spec callouts, and year- or engine-specific caveats.

## Scope

Focus is **1999 model year** and the **2.2L four-cylinder** in factory configuration. Many jobs on these trucks overlap other GMT325 years and engines; this repo prioritizes **accuracy for this truck**. If you have a different year or powertrain, treat everything here as a starting point and verify part numbers, torques, and procedures against your own sources.

## Disclaimer

This is **not** official General Motors documentation. Use this information at your own risk for any maintenance or repair decision. Individual guides should cite or link sources where possible.

## Where this is headed

Over time, expect how-tos, gotcha notes, and parts or reference snippets. Material from the web will be **summarized and attributed**, not copied wholesale. Folder or issue conventions (for example how-to vs. gotcha vs. parts reference) can settle in as the first real docs land.

## Contributing

Corrections and additional **sourced** notes are welcome via issues or pull requests—especially from other **1999 S10** owners who want a single place to park reliable, configuration-specific knowledge.
