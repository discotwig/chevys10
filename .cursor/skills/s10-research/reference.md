# S10 2.2L research — reference bookmarks

Use these as **starting points** for search and reading; verify everything against current threads and your own manuals. **2.2L / four-cylinder** context only.

## Community (search within these sites)

- **S10 forums** — Large GMT325/S10 community; filter threads and posts for **2.2** / **four cylinder**.
- **GM / truck boards** — General Motors truck sections often have S10/Sonoma subforums; **always** narrow to 2.2L.

## Manuals and OEM-style documentation

- **Helm / factory service manual** — Chevrolet S10 pickup (model year–appropriate); use for **torque, fluids, wire diagrams, routing** on the **2.2L** sections.
- **Chilton / Haynes** — Supplemental; cross-check critical torques against factory data when possible.

## Search tips

- Combine: `S10` + `2.2` or `2.2L` + symptom or part (e.g. `electric fan`, `mechanical fan delete`, `cooling`).
- Prefer threads that state **engine** in the title or early posts.

## Note

Forum names and site structures change. If a bookmark is stale, use **web search** with the same **2.2L-biased** terms from [SKILL.md](SKILL.md).
